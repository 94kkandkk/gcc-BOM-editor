# gcc-BOM-editor
GCC 专属产品物料清单（BOM）树状编辑器，聚焦零部件及子零件层级管理、信息编辑，支持层级创建、信息维护与结构可视化，轻量高效管理个人产品全零部件体系。  
A GCC-exclusive BOM tree editor for product parts, focusing on hierarchical management and info editing of parts & sub-parts, supporting hierarchical creation, info maintenance and structure visualization, efficiently managing the full part system of personal products.

---

## 项目简介 / Project Introduction
轻量级树状结构工具，专为个人产品零部件及物料清单（BOM）信息管理设计，可快速搭建零部件层级关系、编辑各节点详细信息，实现产品零部件体系的可视化管理与信息归档。操作简洁，适配各类产品零部件管理需求。  
A lightweight tree-structured tool designed for managing personal product parts and BOM information. It enables quick construction of hierarchical relationships, editing of detailed node information, and visualization of the entire product part system. Simple to use and adaptable to various product part management needs.

---

## 核心功能 / Core Features
✅ 零部件树状层级创建 / 编辑 / 删除  
✅ 子零件节点信息新增与维护  
✅ BOM 结构可视化展示  
✅ 个人产品零部件体系全流程管理  

✅ Create / Edit / Delete hierarchical BOM tree structure of parts and components  
✅ Add and maintain detailed information for sub-part nodes  
✅ Visual display of the entire BOM hierarchy  
✅ Full-process management of personal product part and component system  

---

## 使用说明 / Usage Instructions
- 直接在 GitHub 网页端在线编辑项目源码，无需本地部署  
- 按产品零部件层级关系，逐步搭建 BOM 树状结构  
- 为每个零部件 / 子零件节点完善相关信息，实时保存即可  

- Edit the project source code directly on GitHub Web Editor, no local deployment required  
- Build the BOM tree structure step by step according to the hierarchical relationship of product parts and components  
- Complete relevant information for each part / sub-part node and save in real time
